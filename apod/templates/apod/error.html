Quick Setup Guide

1 Open your terminal in the project directory
2 Fetch a recent APOD (use a date from 2024 or early 2025):
python manage.py fetch_apod --date 2024-12-01
3 Or backfill historical data (recommended - 30 days):
python manage.py fetch_apod --backfill --days 30
4 Populate celestial coordinates:
python manage.py populate_coordinates
Note: NASA's APOD API only provides data up to the current real date. If you're seeing this error, the system date might be set incorrectly, or you need to fetch historical data.
